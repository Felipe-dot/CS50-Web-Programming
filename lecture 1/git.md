# Git Basic Commands

  1. ` git clone <url> `
  2. ` git add <filename> `
  3. ` git commit -m "message" `
  4. ` git status `
  5. ` git push `
  6. ` git commit -am "message" `
  7. ` git pull `
  8. ` git log `
  9. ` git reset --hard <commit> `
  10. ` git reset --hard origin/master `
  11. ` git branch `
  12. ` git checkout -b <branch_name> `
  13. ` git merge <branch_name>`
